# 📊 Task 1 – Digital Literacy Awareness Infographic

This task involves creating a one-page infographic to explain the concept of digital literacy and its importance for students. The infographic was designed using an online tool and covers key topics such as safe internet practices, useful digital tools, and maintaining a professional online presence.

The final design has been exported as an image and added to this folder. This task helped in understanding how to present information visually in a clear and engaging way.
